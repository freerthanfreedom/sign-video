# 화장품 글로벌 런칭 — 마케팅 방향 4가지

직업군 기반 5인 패널(리테일 마케터 제이, 문학 작가 하윤, 순수과학 연구자 닥터 서, 모더니즘 예술가 무나, 역사덕후 반고)의 2라운드 회의(제안 10건 → 교차 지지·기각·조합) 결과.

## 방향 1. 공개 실험실 브랜드 — "완성품이 아니라 과정을 판다" (핵심 축)

회의 최다 지지 수렴점. 닥터 서의 「오픈 랩노트」(실패 이력 공개 연재) + 제이의 「미완성 출시」(소량 배치 + 구매자 투표) + 무나의 「무제」(구매자 명명)를 하나의 사이클로 결합.

- 색조를 소량 '배치 001'로 출시하고 폐기된 처방·실패 기록을 연구노트로 공개. "우리는 23번 실패했고, 그 기록이 여기 있다."
- 구매자 투표와 명명으로 다음 배치를 확정, 탈락 배치는 실패 아카이브로. 완판 속도 데이터가 오프라인 입점 협상 카드가 됨.
- 반고의 반론 수용: 검색성·상표권을 위한 공식 제품 코드는 유지, 소비자 명명은 부제로.

## 방향 2. 지움의 미술관 — 바르는 순간이 아니라 지우는 순간을 판다 (감성·바이럴 축)

무나의 「닦아낸 얼굴」(화장 지운 화장솜 = 하나뿐인 자화상 전시) × 하윤의 「지워지는 문장들」(물에 닿으면 사라지는 잉크 고백 문장) 융합. 클렌징 라인을 브랜드 시그니처로 격상. 화장솜 자화상에 '그날 지운 문장'을 캡션으로 붙이는 고백 아카이브 전시. 자정 전광판은 기각(전환 없는 고정비).

## 방향 3. 검증이 곧 광고 — 피어 리뷰 언박싱 (신뢰 축)

협찬 인플루언서 대신 이해관계 없는 전공자 패널에게 제품과 실험 데이터를 공개 심사받고(편집권은 심사자에게), 반박을 견딘 주장만 광고 카피로 승격하는 규칙을 명문화. "우리를 믿어라"가 아니라 "우리를 의심해라". 역사덕후의 근거: 검증의 형식을 갖춘 지식만 살아남았다(규합총서 vs 납 백분·라듐 크림).

## 방향 4. 리필 재방문 루프 (유통 축)

매대를 사는 대신 유통사에 리필 스테이션 집기를 무상 공급. 재방문이 구조로 강제되고, NFC 태깅으로 고객 데이터가 유통사가 아닌 브랜드에 쌓임. 리필 3회차에 색조 미니어처 증정으로 스킨케어→색조 크로스셀. 리필 위생 규제 고려해 초기 2~3개국 한정.

## 기각된 안

- 복원 시리즈(고문헌 레시피 복원): 기각 2표 — 납·수은 등 독성 사료의 자연주의 오류 리스크(과학), 교양 있는 장식으로 전락(예술). 다큐 콘텐츠 소재로만 보류.
- 타임라인 미러(시대별 미인상 매칭 필터): 휘발성 — 한 번 웃고 끝나는 놀이, 구매로 이어질 다리 부재.
- 자정 전광판: 전환 0의 고정비.
